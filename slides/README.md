# Sub Rosa — Pitch Deck

The presentation deck for the Qualcomm × Meta ExecuTorch Hackathon.

- **[`Sub_Rosa_Deck.html`](Sub_Rosa_Deck.html)** — the deck as an **interactive, animated slideshow**.
  Open in any browser: one slide at a time, scaled to the window, with slide transitions and staggered
  content reveals. Navigate with **← →**, **space**, or **click** (left edge = back); **F** = fullscreen.
  Self-contained (fonts, wax-seal emblem, and the themed Mermaid diagrams embedded — no network needed).
- **[`Sub_Rosa_Deck.pdf`](Sub_Rosa_Deck.pdf)** — the same 13 slides as a flat 16:9 PDF, for sharing or
  printing. (The HTML still prints cleanly to PDF too — the slideshow chrome is screen-only.)

Built in the app's **"Confidential Dossier"** theme (Fraunces / Newsreader / JetBrains Mono;
oxblood-on-paper). The six **architecture slides** embed the live **Mermaid** diagrams from
[`../ARCHITECTURE.md`](../ARCHITECTURE.md), themed to the palette and meant to be shown alongside the
live demo (each carries an "On screen →" cue).

**Outline:** Privacy (the gap) → who carries secrets (→ lawyers) → what Sub Rosa solves → the demo →
architecture I–VI (pipeline · flow · capture+diarization · notes · models/NPU · privacy) →
business model → roadmap → thank you.
